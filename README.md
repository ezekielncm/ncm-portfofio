# Portfolio — W. Ezekiel NACOULMA

Portfolio personnel de **W. Ezekiel Nacoulma**, étudiant en Licence Technologie du Génie Informatique à l'Université Aube Nouvelle (Ouagadougou, Burkina Faso).

## 🌐 Aperçu

Ce site vitrine présente mon parcours, mes compétences, mes diplômes et mes projets. Il est conçu en **HTML / CSS / JavaScript** pur (aucun framework front-end) et utilise les bibliothèques suivantes :

| Bibliothèque | Version | Rôle |
|---|---|---|
| [Particles.js](https://vincentgarreau.com/particles.js/) | 2.0.0 | Animation de particules en arrière-plan |
| [Vanilla Tilt](https://micku7zu.github.io/vanilla-tilt.js/) | 1.7.2 | Effet 3D au survol des cartes |
| [AOS](https://michalsnik.github.io/aos/) | 2.3.1 | Animations au défilement |
| [Font Awesome](https://fontawesome.com/) | 6.2.1 | Icônes |

## 📂 Structure du projet

```
ncm-portfofio/
├── README.md
├── ncm-portfofio.code-workspace
└── public/
    ├── index.html                 ← Page principale
    └── assets/
        ├── css/style.css          ← Feuille de styles
        ├── js/main.js             ← Script principal
        ├── images/                ← Photos & illustrations
        └── pdf/                   ← CV et diplômes (PDF)
```

## 🚀 Démarrage rapide

Aucune étape de build n'est nécessaire. Ouvrez simplement `public/index.html` dans un navigateur ou déployez le dossier `public/` sur n'importe quel hébergement statique (GitHub Pages, Netlify, Vercel…).

## ✨ Fonctionnalités

- **Mode sombre** — Bascule jour / nuit avec persistance (localStorage)
- **Responsive** — Menu hamburger et grilles adaptatives
- **Animations** — Particules, tilt 3D, barres de progression animées, AOS
- **Sections** — Accueil · Compétences · Parcours · Diplômes · Projets · Contact
